# Nexora v1.0 - система мониторинга в реальном времени

**Стек:** Go, React, Electron, WebSocket  

Nexora создан для тех, кто управляет системами, отлаживает приложения или следит за ресурсами в реальном времени.

## Решаемые задачи

### Мониторинг ресурсов в реальном времени
- Графики загрузки CPU и использования оперативной памяти  
- Обновление данных без перезагрузки страницы

### Управление процессами
- Таблица активных процессов с возможностью фильтрации  
- Фильтрация по PID, имени процесса, используемому порту, а также по диапазонам нагрузки CPU и ОЗУ  
- Принудительное завершение процессов

### Сетевой анализ
- Отображение всех занятых сетевых портов  
- Связь процессов с их сетевыми соединениями  
- Запуск нового сетевого процесса с выбором порта через удобное окно

### Интуитивный интерфейс
- Единая панель для мониторинга, фильтрации и управления  
- Мгновенный доступ к ключевой информации о системе

## Четыре специализированных интерфейса

## Технические особенности

Nexora построен на простой, надёжной архитектуре без избыточности — только то, что нужно для мониторинга и управления.

### Единое ядро
Весь системный сбор данных реализован в одном кроссплатформенном бинарнике — для веба и Electron.

### WebSocket в реальном времени
Графики и таблицы обновляются мгновенно через WebSocket-соединение с бэкендом.

### Без эскалации привилегий
Приложение работает от имени текущего пользователя — без запросов sudo или administrator-доступа.

### Linux и Windows
Полная поддержка Linux и Windows. Единый бинарный файл для обеих платформ.

### Запуск процессов
Возможность запустить любой процесс с указанием команды, аргументов и рабочей директории.
